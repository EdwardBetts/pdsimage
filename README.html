CraterInspector

Library to easily access part of the PDS IMAGES from LOLA/LROC_WAC Experiment. Given a location and a radius, for instance the latitude, the longitude and the radius of the crater, this library allows to easily extract both Topography (LOLA) and WAC images of the desired area.

Example

Let's say, we want to get some idea about the crater Copernicus.

C = Copernicus('n','Copernicus',racine,'Crater')
C.Overlay(True)

If we want to focus only on the Central Peak though,

C.Taille_Window = 0.4*C.Diameter
C.Overlay(True)

Contain:

  • CraterInspec:

    • PDS_Extractor : Classes to parse .IMG file from the PDS
    • Structure : Usefull library to visualize the desired structure

  • Data:

    • Data_Crater.csv : Comprehensive dataset of lunar impact craters.

      • Source of the dataset
        • Salamuniccar et al 2011
        • Jozwiak et al 2015
        • Losiak et al 2009
      • Columns
        • Name (when given), Lat (degree) , Long (degree) , Diameter (km) + Type (1 = Floor-fractured crater, 0 = Normal impact craters)
        • A comprehensive description of FFC can be found here

    • Data_Dome.csv

  • PDS_FILE

    • Binary image .IMG files to download on the PDS node
    • Library optimize for LOLA and LROCWAC Images (FOLDERs inside PDSFILE). Nevertheless, It could be easily extended to account fo other type of data.
